<h1>Web Programming Homework Demo</h1>

<p>This project uses HTML,CSS,JavaScript and PhP to implement a home service system. System consists of 2 parts: First part is producer. This one is the admin
and there can only be one producer. Admin can change everything around house such as lights, temperatures, music and send mails or add events to calendar or lock doors.
Second part is consumer part. There could be multiple consumers. They can see data through the interface but they can't change anything.</p>
